O try-catch é uma estrutura usada para lidar com exceções em tempo de execução.
No bloco try, colocamos o código que pode gerar erros.
No bloco catch, definimos como o programa deve reagir caso o erro aconteça.
Essa abordagem não impede que o erro ocorra, mas previne que ele interrompa o funcionamento do sistema, garantindo maior estabilidade e confiabilidade. Assim, o software consegue tratar falhas de forma controlada e oferecer uma melhor experiência ao usuário.
